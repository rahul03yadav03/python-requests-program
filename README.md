- This project contains simple Python programs created to practice working with websites and APIs using the requests library.
- It is designed for beginners and undergraduate students who want to understand how HTTP requests work and how to interact with web servers using Python.

# OBJECTIVES

- Learn how HTTP/HTTPS requests work
- Understand status codes and server responses
- Practice handling redirects and headers
- Work with JSON data from APIs
- Measure website response time
- Handle request errors properly

# TASKS IMPLEMENTED


- Check if a website is reachable
- Display HTTP status codes
- Detect whether a website supports HTTPS
- Identify and display redirect chains
- Measure total response time
- Extract and display HTTP response headers
- Save website HTML content into a file
- Preview the first 200 characters of a webpage
- Fetch and display JSON data from APIs
- Check multiple URLs in one run
